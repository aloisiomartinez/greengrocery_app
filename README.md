
# GreenGroceryApp

App de loja online, nele é possível fazer autenticação, buscar produtos e adicionar ao carrinho, realizar pagamento via Pix.



## Tecnologias usadas
- Flutter
- Consumo de REST API's com Dio
- GetX
- Freezed
- Build_runner
- Json_serializable
